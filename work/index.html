signals

Signals should only be used if warrented from MUTCD. Only be used if agreed upon by a Traffic Engineer. Only be used if will not conflict with any safety conflicts.

Signals are the primary solution for heavy traffic at intersections > 45,000 AADT

A roundabout can only accomadate ~1,000 vphpl. If multiple lanes are needed carefull design vehicle turning analysis to be done, in addition to the appropriate ROW
